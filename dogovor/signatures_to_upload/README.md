# Подписи оптометристов

Имена файлов должны совпадать с фамилиями оптометристов:

| Файл | ФИО |
|------|-----|
| `Карташева.png` | Карташева Татьяна Владиславовна |
| `Величко.png` | Величко Ирина Владимировна |
| `Еремина.png` | Еремина Анжелика Юрьевна |
| `Ефремова.png` | Ефремова Светлана Павловна |
| `Ивченко.png` | Ивченко Виолетта Евгеньевна |
| `Никитина.png` | Никитина Елена Михайловна |

Загрузка в Supabase (из корня репозитория):
```bash
cd dogovor/backend
.venv/bin/python -m app.services.signature_upload ../signatures_to_upload
# или без аргумента (скрипт возьмёт dogovor/signatures_to_upload):
# .venv/bin/python -m app.services.signature_upload
```
