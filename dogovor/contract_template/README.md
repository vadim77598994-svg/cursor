# Шаблон договора (.docx)

Официальный шаблон договора в формате Word. **Без персональных данных** — безопасно хранить в git.

- **Имя файла:** `ПАЙ_ОПТИКС_МЕД_ДОГ.docx`
- Положите файл в эту папку один раз (или он уже в репозитории).

После правок в .docx запустите из папки `dogovor/scripts`:

```bash
python3 sync_from_docx.py
```

Скрипт извлечёт текст в `backend/templates/contract_extracted.html` для сравнения с `contract.html`.
