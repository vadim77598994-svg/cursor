# Где в договоре стоят подписи (п.3 — скриншоты)

Вы присылали скриншоты с красными стрелками: «подпись оптометриста» и «подпись пациента». По ним уже всё сделано в коде.

## Что сделано по скриншотам

1. **Подпись оптометриста** (картинка из БД, `staff_signature_url`) подставляется в **3 места**:
   - основной блок «РЕКВИЗИТЫ И ПОДПИСИ СТОРОН» (строка с «Оптометрист _______/ {{ staff_fio }}»);
   - приложение 1 (Информированное согласие) — «Исполнитель _______/ {{ staff_fio }}»;
   - приложение 3 (Акт сдачи-приёмки) — «Исполнитель _______/ {{ staff_fio }}».

2. **Подпись пациента** (рисованная на планшете, `patient_signature_data_url`) подставляется в **4 места**:
   - тот же основной блок — «Подпись Пациента: …»;
   - приложение 1 — «Пациент _______/ {{ patient_fio }}»;
   - приложение 2 — «Пациент _______/ {{ patient_fio }}»;
   - приложение 3 — «Пациент _______/ {{ patient_fio }}».

В шаблоне это реализовано так: если переменная с подписью есть (например `patient_signature_data_url`), рендерится `<img src="...">`, иначе остаётся прочерк.

## Итог

Скриншоты учтены: места из разметки совпадают с тем, что в коде. Дополнительных правок по скриншотам не требуется. Если понадобится добавить подпись ещё где-то — напишите, в каком именно месте договора (страница/приложение и строка).
