# Напоминания по проекту «Договор»

- **Подписи оптометристов.** ✅ Реализовано: подписи загружены в Storage, в PDF подставляются по ФИО; отчёт для сверки — `docs/SIGNATURE_VERIFY_REPORT.md`. *(Изначально: заказчик присылает подписи → загрузка в Storage и `signature_image_url` в `dogovor_staff`.)*

- **OCR.** Сейчас используется **Beorg** (разворот 2–3 + прописка), не Tesseract. Учтено: iPad/Safari, камера + галерея, индикатор загрузки и таймаут, подсказки по освещению, данные не хранятся дольше нужного. Ручной ввод и «ввести вручную» доступны. Подробности — `docs/OCR_КАК_СДЕЛАТЬ.md` (общая логика), реализация — `backend/app/services/beorg_recognize.py`.
