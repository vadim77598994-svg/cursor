# Напоминания по проекту «Договор»

- **Подписи оптометристов.** Перед реализацией вставки подписи оптометриста в PDF договор — напомнить заказчику прислать отсканированные подписи каждого оптометриста. После получения загрузить в Supabase Storage и проставить `signature_image_url` в таблице `dogovor_staff` для каждого сотрудника.

- **OCR (Tesseract.js).** При реализации шага «Сканирование» учесть: работа на iPad в Safari, камера + выбор из галереи, индикатор загрузки 5–15 сек, подсказка по освещению, не хранить фото >30 сек, парсер с запасом на «плывущий» вывод Tesseract. Подробно — `docs/OCR_КАК_СДЕЛАТЬ.md`.
