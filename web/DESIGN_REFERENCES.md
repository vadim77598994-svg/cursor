# Референсы для визуала и UX

Использовать при вёрстке галереи и общем стиле сайта.

---

## 1. [Nothing Playground](https://playground.nothing.tech/)

- **Тема:** тёмная, минималистичная.
- **Секции:** блоки по типам контента (Essential Apps, Glyph Toys, Camera Presets и т.д.); у нас — можно секции по редкости или типу оправы.
- **Карточки:** название, автор/счётчик, превью; лаконично, без перегруза.
- **Типографика:** чёткая, читаемая; акцент на заголовках секций.
- **Общее:** «soft film», спокойная сетка, много воздуха.

---

## 2. [Teenage Engineering](https://teenage.engineering/)

- **Тема:** светлая/нейтральная, очень минималистичная.
- **Сетка:** продукты в сетке, у каждого — изображение и короткий CTA («buy now»); у нас — фото очков + название/атрибуты, при желании кнопка «Подробнее» / «Передарить».
- **Типографика:** крупная, смелая; моно или геометрический шрифт.
- **Общее:** много пустого пространства, акцент на продукте, без лишних декораций.

---

## Для галереи NFT Glasses (реализовано)

- **Тема:** тёмная (#0a0a0a), гибрид Nothing + TE: тёмный фон, крупная типографика.
- **Шрифт:** Space Grotesk (next/font) — геометрический, читаемый.
- **Секции (Nothing):** при нескольких редкостях — блоки по типу (Legendary, Rare, …), заголовки секций uppercase + letter-spacing.
- **Карточки:** фото 45:55, название (крупнее), редкость · линзы · оправа; много отступов (gap-10/12, py-16/24).
- **Фильтры:** чипы, uppercase «Filter», кнопка «clear»; минималистично.
- **Деталь:** крупный заголовок, атрибуты с uppercase лейблами, блок «Передарить» с закруглением и воздухом.
